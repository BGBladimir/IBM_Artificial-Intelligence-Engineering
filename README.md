# IBM_Artificial-Intelligence-Engineering

"Este curso aborda conceptos fundamentales del aprendizaje automático y el aprendizaje profundo, que incluyen tanto el aprendizaje supervisado como el no supervisado, utilizando lenguajes de programación como Python. Se utilizan bibliotecas populares de aprendizaje automático y aprendizaje profundo como SciPy, Scikit-Learn, Keras, PyTorch y TensorFlow para abordar problemas de la industria relacionados con el reconocimiento de objetos, visión por computadora, procesamiento de imágenes y video, análisis de texto, procesamiento del lenguaje natural (NLP), sistemas de recomendación y otros tipos de clasificadores."
